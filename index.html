Moon Rising App

This app allows the user to gather real-time data about the Earth's moon.

The Moon Rising App uses vanilla JavaScript to make an AJAX call in order to request JSON data from a RESTful API. This app was created using JavaScript, HTML5, and CSS3. The app is responsive.
